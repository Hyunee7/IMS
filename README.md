# 개요
Javascript 로 Adlib 연주프로그램 IMPlay 의 사운드파일 IMS 내용을 보는(?) 웹페이지 제작 프로젝트

# 기능
Adlib 용 연주파일 IMS 를 읽어 OCPlay 처럼 화면에 재생구현을 함.
OPL기능을 구현하지 못해 사운드는 재생되지 못함.

# Files
- *.IMS : 연주파일
- *.ISS : 가사파일
- *.BNK : 연주를 위한 악기 모음 파일
- kssmcode.js : GetByte 모듈
- ims.html : IMS 제목, 악기, 볼륨메타, 컴포저 보는 웹페이지(Javascript 포함)
